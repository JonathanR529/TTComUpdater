# TTComUpdater
A simple updater for the TeamTalk Commander

I wrote this utility because a few friends of  mine and I wanted an easy way to update TTCom, without downloading the zip file each time, and this achieves that!

Disclaimer! This is only for Windows, because there is no way to run batch scripts on Mac OS.

Paste all files in this folder except readme.md into your TTCom directory, and then run the TTCom updater.bat file.

Just run the bat file, and it will explain everything to you. The keys are listed below because cmd will not always read the keys to you.

## Keys

y: Yes to the update prompt.

n: No to the update prompt.

b: Updates to the latest beta revision of TTCom.
